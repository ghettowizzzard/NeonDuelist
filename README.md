# NeonDuelist
A dueling game based upon Popular TCG's with a fun and unique twist! Adding Multiplayer Servers into the Game.

Setup Instructions:

1) Download all 3 files in the repository (or as a ZIP).

2) Place them all inside of one Folder together.

3) Run NeonDuelist (either by double clicking > or dragging into your browser)

4) Enjoy the game!
